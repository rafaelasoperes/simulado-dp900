13. Azure Data Studio can be used to query an Azure SQL database from a device that runs macOS.  SIM não
14. Microsoft SQL Serve Management Studio (SSMS) enables users to create ande use SQL notebooks.  sim NÃO
15. Azure Data Studio can be used to restore a database.  SIM não

Tradução

#13. O Azure Data Studio pode ser usado para consultar um banco de dados SQL do Azure de um dispositivo que executa o macOS. SIM não
#14. O Microsoft SQL Serve Management Studio (SSMS) permite que os usuários criem e usem notebooks SQL. sim NÃO
#15. O Azure Data Studio pode ser usado para restaurar um banco de dados. SIM não

Explicação

Caixa 1: Sim -
O Azure Data Studio é uma ferramenta de banco de dados multiplataforma para profissionais de dados que usam plataformas de dados locais e na nuvem no Windows, macOS e Linux.
Você pode usar o Azure Data Studio para se conectar a um servidor do Banco de Dados SQL do Azure. Em seguida, você executará instruções Transact-SQL (T-SQL) para criar e consultar bancos de dados SQL do Azure.

Caixa 2: Não -
O SQL Server Management Studio serve para configurar, gerenciar e administrar todos os componentes no Microsoft SQL Server, não para criar blocos de anotações SQL.
Em vez disso, use o Azure Data Studio para criar o notebook SQL.

Caixa 3: Sim -
Você pode usar o Azure Data Studio para restaurar bancos de dados.
